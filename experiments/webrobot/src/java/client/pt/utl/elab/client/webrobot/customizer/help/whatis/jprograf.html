O JPrograf é um clone do Prograf programado em java.

Este programa permite programar um robot, através do recurso a diversos blocos.

Para programar o robot não necessita de conhecimentos de nenhuma linguagem de programação, dado que os blocos são depois interpretados por uma máquina de estados que existe no servidor da experiência.

O JPrograf foi programado por André Neto, como parte do seu projecto final de curso.
