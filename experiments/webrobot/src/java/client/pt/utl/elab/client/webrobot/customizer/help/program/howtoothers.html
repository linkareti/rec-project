• Gravar um ficheiro

Seleccionar no menu Arquivo uma das seguintes opções:

-Guardar;

-Guardar como...

Os ficheiros são gravados com a extensão .grf e podem ser lidos pelo JPrograf e pelo Prograf (versão 1.4 ou superior).

  • Abrir um ficheiro

Através da opção abrir no menu Arquivo.

  • Imprimir ficheiros

Opção imprimir no menu Arquivo

  • Anular/Refazer

Opção Anular ou Refazer no menu editar.
