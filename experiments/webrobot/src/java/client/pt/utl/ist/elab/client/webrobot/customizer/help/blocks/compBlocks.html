Bloco de comparação de números inteiros

Este bloco permite verificar se uma variável é maior, menor ou igual a:

-outra variável;

-um número inteiro entre 0 e 255.

 

Bloco de comparação de binária

Verifica o estado lógico de uma variável:

Bloco de comparação de comparação AND

-Bloco de comparação de comparação números inteiros AND

-Bloco de comparação de binária AND

Estes tipos de blocos são muito semelhantes aos blocos de comparação simples, permitindo no entanto o encadeamento de vários blocos de comparação.

No caso de um conjunto de vários blocos AND todas as condições do conjunto têm de ser verificadas para passar ao próximo bloco a seguir ao conjunto, caso contrário segue para o bloco à direita do primeiro bloco do conjunto.
